### Employee Management System (HRMS)

A comprehensive HRMS application built with Angular for managing employee workflows across an organization.  
I contributed to the frontend development of multiple functional areas, ensuring smooth user experience, modular architecture, and maintainable code.

**Project Description:**  
This HRMS system enables companies to manage employee attendance, leaves, timesheets, project activities, team overviews, and daily productivity. The application focuses on real-time updates, clean UI, and seamless API integration.

**Key Features:**

- Real-time dashboard with HR insights
- Punch In / Punch Out attendance workflow
- Live punchboard updates
- Timesheet tracking
- Leave application & approval workflows
- Project activity tracking
- Integrated calendar
- Team overview showcasing employee summaries
- Responsive UI built with PrimeNG

**Tech Stack:** Angular, TypeScript, SCSS, PrimeNG, RxJS, REST API, Jasmine

### Kesher – CRM System

A CRM solution built with Angular and Bootstrap to help businesses manage customer relationships, sales activities, and operational workflows.
